# SyntecxHub Project - Hand Gesture Recognition

## 📌 Overview

Hand Gesture Recognition is a Computer Vision project developed using Python, OpenCV, and MediaPipe. The system detects hand landmarks in real time through a webcam and recognizes predefined gestures such as Open Palm, Fist, and Thumbs Up.

The project demonstrates how hand gestures can be used as a natural and interactive way to communicate with computer systems.

## 🎯 Objective

To develop a real-time hand gesture recognition system capable of detecting hand landmarks and classifying basic gestures using Computer Vision techniques.

## ✨ Features

* Real-time hand detection and tracking
* Hand landmark recognition using MediaPipe
* Gesture classification:

  * Open Palm
  * Fist
  * Thumbs Up
* Live webcam feed with gesture display
* Efficient and lightweight implementation

## 🛠 Technologies Used

* Python
* OpenCV
* MediaPipe

## ⚙ Installation

Install the required libraries using:

```bash
pip install opencv-python mediapipe
```

## ▶ Usage

Run the Python file:

```bash
python hand_gesture_recognition.py
```

The webcam will open and start detecting hand gestures in real time. Press **Q** to exit the application.
